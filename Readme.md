# Astar 설명을 위해 만든 페이지입니다.
====================================

### 사용안내
> 1. Typescript를 이용해서 만들어졌습니다. 
> 2. 사이트를 보시려면 다음 링크를 접속하면 됩니다. <br>
    https://gondr99.github.io/astarweb/
> 3. 코드는 자유롭게 이용하셔도 됩니다. 


### 사용방법

패키지 설치
```
npm install 
```

Typescript구동 
```
npm run watch
```

이후 라이브 서버등으로 index.html 을 보시면 됩니다.
